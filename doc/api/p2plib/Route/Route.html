Creates a new Route instance.

peerId The unique identifier of the peer. canForward Whether the peer is capable of forwarding messages to other peers. Defaults to false. header An optional PacketHeader to add to the route's header history. addresses An optional map of FullAddress to AddressProperties to initialize the route's known addresses for the peer. address An optional address and its properties to add to the route. This is a convenience parameter for adding a single address.
