PacketHeader

PacketHeader class

Structure of a packet header:

  • 1 byte: forwards count
  • 1 byte: packet type
  • 6 bytes: issuedAt (Unix timestamp with milliseconds)
  • 8 bytes: message id (int)

Total header length: 16 bytes Represents the header of a network packet.

This class encapsulates the metadata associated with a packet, including its type, ID, timestamp, and forwarding information. It is immutable, meaning its properties cannot be changed after creation.
