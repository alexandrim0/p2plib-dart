Creates a new RouterBase instance.

crypto The cryptography instance to use for encryption and signing. If not provided, a default Crypto instance is created. transports The list of transports to use for network communication. If not provided, default UDP transports for IPv4 and IPv6 are created. messageTTL The time-to-live for messages, specifying how long they should be considered valid. Defaults to 3 seconds. keepalivePeriod The interval at which keepalive messages are sent to maintain connections with peers. Defaults to 15 seconds. logger The logger to use for logging events.
